# Pong Game

A classic Pong game built with HTML, CSS, and JavaScript.

## Features

✨ **Player vs Computer** - Control your paddle with arrow keys or mouse
🎮 **Smooth Controls** - Dual input support (keyboard + mouse)
🏓 **Ball Physics** - Realistic bouncing with spin mechanics
🤖 **AI Opponent** - Challenging computer player
📊 **Score Tracking** - Live scoreboard for both players
✅ **Collision Detection** - Accurate paddle and wall collision system
🎨 **Modern UI** - Beautiful neon-style design
📱 **Responsive** - Works on different screen sizes

## How to Play

1. **Open** `index.html` in your web browser
2. **Click** "Start Game" to begin
3. **Control your paddle** (left side) using:
   - **Arrow Keys** (↑ ↓)
   - **Mouse** movement
4. **Score points** when the computer misses the ball
5. **Click** "Pause Game" to pause or "Reset Score" to start over

## Game Mechanics

- **Paddles**: Move up and down to hit the ball
- **Ball**: Bounces off paddles and walls
- **Scoring**: Point awarded when opponent misses the ball
- **Speed**: Ball gradually increases speed with each paddle hit
- **Spin**: Ball angle changes based on paddle hit position

## Files

- `index.html` - Game structure and layout
- `style.css` - Styling and animations
- `script.js` - Game logic and mechanics

## Controls

| Input | Action |
|-------|--------|
| ↑ Arrow Up | Move paddle up |
| ↓ Arrow Down | Move paddle down |
| Mouse | Move paddle to mouse Y position |
| Start Game | Begin the game |
| Pause Game | Pause/Resume the game |
| Reset Score | Reset scores to 0 |
